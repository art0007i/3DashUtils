# 3DashUtils

A [Bepinex](https://github.com/BepInEx/BepInEx) 5.x plugin for [3Dash](https://delugedrop.itch.io/3dash)

It adds a menu (toggled by pressing tab) which gives you access to a bunch of different utilities and fixes.
